## ğŸ†˜ Troubleshooting

### App Won't Start
**File is blocked:**
1. Right-click the .exe file
2. Select **Properties**
3. Check **Unblock** at the bottom
4. Click **OK**

### GitHub Features Don't Work
**Git is not installed:**
- Download Git: https://git-scm.com/download/win
- Install with default options
- Restart the app

### Antivirus Blocking
**False positive detection:**
- Add exception in your antivirus
- Or download from trusted source

### Data Not Saving
**Check project folder:**
- Ensure you selected a valid folder in Settings
- Folder should have write permissions
- Don't use read-only drives

## ğŸ› Common Issues

**Q: Settings won't open**
A: Check if another dialog is open. Close it first with Esc.

**Q: Can't see FABs**
A: They're in the bottom-right. Check if status bar is blocking them.

**Q: Command Palette not working**
A: Make sure Ctrl+K isn't captured by another app.

**Q: Breadcrumbs not showing**
A: They only show when navigating categories.

**Q: Right-click menu not appearing**
A: Make sure you're right-clicking directly on a product card.
